# Operations Research and Optimization Portfolio

This repository contains a collection of applied Operations Research projects focused on mathematical optimization, synthetic data generation, Python-based model implementation, scenario analysis, and interactive decision-support dashboards.

Each project includes:
- A clear business or operational problem
- A mathematical optimization formulation
- Synthetic input datasets
- Python implementation
- Solver results and interpretation
- Scenario or sensitivity analysis
- Streamlit dashboard for stakeholder interaction
